# 💬 Real-Time Chat App — Frontend

A real-time chat application where users can create or join unique chat rooms instantly. Built with Angular and Socket.io.

🔗 **Backend Repo:** [chat-app-backend](https://github.com/HasanAshee/chat-app-backend)

## ✨ Features
- Create or join any room instantly by name
- Real-time messaging with WebSockets
- Infinite unique rooms — each with its own message history
- Messages persisted in MongoDB

## 🛠 Tech Stack
Angular · TypeScript · Socket.io · RxJS

## 🚀 Getting Started
```bash
npm install
ng serve
```
